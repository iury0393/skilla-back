# Skilla Back-End

Skilla Back-End é feito com Node(Express) + MongoDB.

## Rodando Local

Na raíz do projeto, crie um .env a partir do .env-example e configure como se pede

Então rode <code>yarn && yarn run dev</code> para iniciar o server.
